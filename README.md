Java Web Application (Maven Project)

 Project Overview
This project is a Java-based web application built using Apache Maven for dependency management and build automation.
The goal of this project was to practice building, packaging, and managing a Java web application using modern development tools.

 Technologies Used
- Java
- Apache Maven
- HTML / CSS
- Apache Tomcat
- ec2 instance
- Git & GitHub
